# ADA PWA Demo

## ¿Cómo usar este proyecto?

- Instalar dependencias

```
yarn install
```

- Abrir el navegador en `http://localhost:8080`

## ¿Cómo generar la aplicación lista para producción?

- Instalar dependencias

```
yarn build
```

- En el directorio `dist` se encuentran los archivos compilados
